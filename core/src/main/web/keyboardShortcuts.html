Beaker Keyboard Shortcuts

On input cells

Cell editing Ctrl-C (Win) - Cmd-C (Mac) Copy
Ctrl-X (Win) - Cmd-X (Mac) Cut
Ctrl-V (Win) - Cmd-V (Mac) Paste
Ctrl-Z (Win) - Cmd-Z (Mac) Undo
Ctrl-Y (Win) - Cmd-Y (Mac) Redo
Ctrl-Space (Win) - Cmd-Space (Mac) Code auto-complete
Alt-F11 Toggle fullscreen editor
Esc Exit fullscreen editor
Cell Evaluation: Ctrl-Enter (Win) - Cmd-Enter (Mac) Evaluate
Shift-Enter Evaluate and move focus to next cell.
Notebook editing: Ctrl-Shift-A (Win) - Cmd-Shift-A Add a new code cell
Ctrl-Shift-E (Win) - Cmd-Shift-E Pop up evaluator selector
Ctrl-S (Win) - Cmd-S (Mac) Save notebook
Ctrl-Alt-Up (Win) - Cmd-Alt-Up (Mac) Move cell up
Ctrl-Alt-Down (Win) - Cmd-Alt-Down (Mac) Move cell down
Ctrl-Alt-D (Win) - Cmd-Alt-D (Mac) Delete cell
Ctrl-Alt-Hide (Win) - Cmd-Alt-Hide (Mac) Hide cell
Cell navigation: Alt-Down (or Alt-J) Move focus to next code cell
Alt-Up (or Alt-K) Move focus to previous code cell

On the Start Page

Ctrl-N (Win) - Cmd-N (Mac) Create new empty notebook
Ctrl-Shift-N (Win) - Cmd-Shift-N (Mac) Create new default notebook
