Beaker Keyboard Shortcuts

Input cell

Evaluation: Ctrl-Enter Evaluate
Shift-Enter Evaluate and move focus to next cell.
Autocomplete: Ctrl-Space Code complete
Notebook: Ctrl-Shift-A Add a new code cell
Ctrl-Shift-E Pop up evaluator selector
Ctrl-S Save notebook
Fullscreen: Alt-F11 Toggle fullscreen
Esc Exit fullscreen
Navigation: Alt-Down (or Alt-J) Move focus to next code cell
Alt-Up (or Alt-K) Move focus to previous code cell
